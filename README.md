# Simple Dark Mode Toggle 🌙

## 🚀 Características
- **Centrado Perfecto:** Implementado con Flexbox.
- **Transiciones Suaves:** Cambio de color fluido mediante `transition`.
- **Tematización Limpia:** Uso de `:root` para gestionar colores en un solo lugar.
- **Sin Dependencias:** 100% Vanilla JavaScript.

## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3 (Variables & Flexbox)
- JavaScript (DOM Manipulation)

## 📋 Cómo usarlo
1. Clona este repositorio.
2. Abre el archivo `index.html` en tu navegador.
3. ¡Haz clic en el botón para alternar entre el modo claro y oscuro!
